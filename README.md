## 🗂 How to download and execute the project

---

```bash

    # Clone repository
    $ git clone https://github.com/douglas-bernardo/node-api-boilerplate.git

    # Entry on folder
    $ cd node-api-boilerplate

    # Install dependences
    $ yarn install

    # Run script
    $ yarn dev:server

    # access
    # http://localhost:3333
```
